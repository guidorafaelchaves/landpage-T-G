# T&G Innovations — Landing Page Cinematográfica

Landing page em React + Vite com Three.js, GSAP ScrollTrigger, Lenis e Framer Motion.

## Rodar localmente

```bash
npm install
npm run dev
```

## Gerar produção

```bash
npm run build
```

A pasta `dist/` pode ser publicada no GitHub Pages, Cloudflare Pages ou Netlify.
